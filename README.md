# SentimentScope-Smart-Text-Sentiment-Analyzer
AI-powered sentiment analyzer built with Streamlit and Transformers to detect emotions in text data. Provides real-time insights, aspect-based analysis, and visual sentiment dashboards for smarter decision-making.

# Description
SentimentScope is an AI-powered web application that performs intelligent text sentiment analysis using Natural Language Processing (NLP) and Transformer models. It helps users analyze public opinions, reviews, and social media posts to gain real-time emotional insights. Built with Streamlit and Hugging Face Transformers, it provides an interactive dashboard for visualizing sentiment trends and aspect-based feedback analysis.

# Tech Stack
Frontend/UI: Streamlit
Backend: Python
Libraries: Transformers, Pandas, NumPy, Matplotlib
Model: Pre-trained BERT (Hugging Face)
Dataset: Stanford Sentiment Treebank
Deployment: Localhost / Streamlit Cloud

# Work Flow
1. Upload text data (CSV or manual entry).
2. Text is cleaned and processed using NLP techniques.
3. Transformer model classifies sentiments (Positive, Negative, Neutral).
4. Results are visualized through bar charts and sentiment metrics.
5. Aspect-based breakdown provides deeper feedback insights.
